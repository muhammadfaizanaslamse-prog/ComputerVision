# 🖼️ OpenCV Image Editor

A beginner-friendly **Image Editor application** built using **Python and OpenCV**.

This project is part of my **60 Days Computer Vision Learning Journey**.  
The goal of this project is to understand fundamental image processing operations using OpenCV.

---

## 🚀 Features

### 📥 Image Handling
- Load images using OpenCV
- Display original images
- Save processed images

### 🎨 Image Processing
- Convert images to grayscale
- Apply Gaussian Blur
- Perform Canny Edge Detection

### 🔄 Image Transformations
- Resize images
- Flip images horizontally

---

## 🛠️ Technologies Used

- Python
- OpenCV
- NumPy
